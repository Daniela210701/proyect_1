Proyecto 1: Visión Artificial

Desarrollado por: Ing. Erika Barreto

🖥️ Entorno Local Vinculado (YOLOv8 + Pygame)

Estructura del Sistema (3 Variantes):

🐍 1. interaccion.py (Principal)
Script maestro que ejecuta el bucle de Pygame, procesa los frames de la cámara y despliega la interfaz gráfica interactiva con las predicciones de YOLOv8.
⚙️ 2. prueba_1.py
Entorno de pruebas controlado para la calibración del umbral de confianza (threshold) de las redes neuronales y testeo de cámara.
🛠️ 3. prueeba2_.py
Script secundario optimizado para pruebas de rendimiento de hardware y filtrado de frames por segundo (FPS).

Código fuente protegido y versionado en GitHub.
